A complete authentication system built with modern web technologies that provides secure user authentication and account management features. Users can create an account, log in, manage their profile, update personal information, change their password, and delete their account.

Features:-
    User Registration (Sign Up)
    Secure User Login
    Profile Photo Upload
    User Dashboard
    Update Profile Information
    Logout Functionality
    Delete (deactivate) Account

User Flow:-
    1. User signs up by creating a new account.
    2. User is redirected to the dashboard.
    3. User can:
            View profile details
            Update personal information
            Upload or change profile photo
            Change password
            Logout securely
            Delete account permanently
    4. User can login again using registered credentials (if account not deleted).
